Retención Presuntiva

En este campo obligatorio debe ingresar S si la transacción de venta que registra tiene Retención Presuntiva, y N si no lo tiene. Se aceptará solamente MAYÚSCULAS. Este concepto se utiliza solo para comercializadoras de combustibles.
